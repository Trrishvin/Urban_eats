# TheCoreFive | Urban Eats – Urban Review Platform

**Live Demo:** [https://trrishvin.github.io/Urban_eats](https://trrishvin.github.io/Urban_eats/)

TheCoreFive is a high-end, modern urban food blog and review archive designed with a focus on immersive UI/UX. The platform serves as a digital "Vault" for culinary experiences, featuring advanced CSS techniques, interactive JavaScript components, and a sophisticated dark-aesthetic brand identity.

## 🚀 Key Features

* **Glassmorphism UI:** Built with a modern aesthetic using backdrop-filters, semi-transparent surfaces, and neon-primary accents.
* **Dual-Theme Engine:** Integrated light and dark mode switching with persistent CSS variables for a customized user experience.
* **Interactive Review Vault:** A dedicated archival system for detailed food critiques, featuring specialized "Verdict Boxes" and rating badges.
* **Dynamic UX Elements:** Includes a custom-coded loading sequence, real-time localized clock, and smooth scroll-triggered parallax effects.
* **Social Ecosystem:** A comprehensive, responsive grid connecting the platform across Instagram, YouTube, LinkedIn, Reddit, and more.

## 🛠️ Technical Stack

* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
* **Scripting:** Vanilla JavaScript (ES6+) for theme logic and scroll animations
* **Typography:** Space Grotesk & Syncopate via Google Fonts
* **Icons:** Font Awesome 6.5.1

## 📂 Project Structure

* **index.html:** The main landing page featuring the hero section and social grid.
* **good reviews.html:** The "Vault" containing the full archive of 8 detailed food reviews.
* **logo.jpg:** Custom-designed brand identity for TheCoreFive.
